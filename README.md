# 控制台营救公主小游戏

## 项目简介

这是一个使用 C# 编写的控制台小游戏。玩家需要在地图中移动，找到并击败 Boss，随后营救公主。游戏通过控制台字符绘制地图、玩家、Boss 和公主，并使用键盘输入完成移动、战斗和菜单选择。

游戏主要包含以下内容：

- 开始菜单
- 游戏地图
- 玩家移动
- Boss 战斗
- 公主营救
- 游戏结束界面

---

## 游戏目标

玩家需要完成以下流程：

1. 从地图左上角出发
2. 移动到 Boss 附近
3. 按下攻击键与 Boss 战斗
4. 击败 Boss 后，公主出现
5. 移动到公主旁边并营救公主
6. 成功通关

如果玩家在战斗中生命值归零，则游戏失败。

## 操作说明

### 菜单界面操作

| 按键 | 功能 |
|---|---|
| W | 选择上一个选项 |
| S | 选择下一个选项 |
| J | 确认当前选项 |

### 游戏界面操作

| 按键 | 功能 |
|---|---|
| W | 向上移动 |
| S | 向下移动 |
| A | 向左移动 |
| D | 向右移动 |
| Q | 攻击 Boss / 营救公主 |

## 游戏元素说明

| 图标 | 含义 |
|---|---|
| ■ | 墙体 |
| ▲ | 玩家 / Boss |
| ● | 公主 |

说明：

- 玩家使用黄色 `▲` 表示
- Boss 使用绿色 `▲` 表示
- 公主使用蓝色 `●` 表示
- 红色 `■` 表示地图边界和分隔墙

## 游戏规则

### 1. 玩家移动

玩家可以通过 `W`、`A`、`S`、`D` 控制移动。

由于控制台字符宽高比例不同，横向移动时每次移动 2 个字符，纵向移动时每次移动 1 行。

玩家不能穿过墙体，也不能直接走到 Boss 或公主所在的位置。

---
